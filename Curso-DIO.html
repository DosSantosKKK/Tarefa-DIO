Oque são TAGS(voltar)

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros ou seja, quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

Sabendo disso, usamos algumas Tags aprendidas durante essa etapa para a criação dessa pagina

Tags usadas para fazer essa pagina(voltar)

  • Head - Utilizada para definir o começo e o fim do cabeçalho de um documento HTML
  • Body - Indica para o navegador o corpo do documento HTML, ou seja, a parte principal da página web
  • H1, H2... - Usada para definir os titulos de cada paragrafo
  • Strong - Usada para deixar o texto em negrito
  • U - Usada para deixar a parte selecionada com a linha inferior "underline"
  • A href - Esta Tag é usada para referenciar certa parte do texto que pode ser usada para redirecionar até o local escolhido
  • I - Usada para deixar o texto em italico
Entre diversos outras tags.
